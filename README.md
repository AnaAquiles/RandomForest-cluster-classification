# 🌲 RandomForest Cluster Classification

A simple but usefull project to explore the relationship between **anatomy** and **function** through spatial clustering.

## 🧠 What’s this about?

This project uses a **Random Forest classifier** to analyze and categorize clusters based on how they are distributed in space. The goal? To uncover patterns that might link structural (anatomical) properties with functional behavior.

## 🔍 Features Used

Each cluster is described using a few intuitive geometric properties:

- **Cluster Size** → Total number of cells  
- **Ellipse Length** → The major axis (how stretched the cluster is)  
- **Ellipse Area** → Overall spatial footprint  

These features help the model "understand" how clusters differ from one another.

## 🚀 Why it’s cool

- 🧩 Turns spatial patterns into meaningful classifications  
- 🌐 Bridges the gap between structure and function  
- ⚡ Lightweight and easy to experiment with  

## 🛠️ Use Cases

- Exploring functional (time-series) with (real coordinates) anatomical features  
- Identifying spatial organization patterns  
- Quick prototyping for clustering-based classification tasks  

## ✨ TL;DR

Feed in cluster geometry → get meaningful classification → discover hidden relationships.
